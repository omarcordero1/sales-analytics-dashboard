# Sales Intelligence Dashboard 🚀

**Análisis de ventas B2B con Python + Streamlit | Omar Cordero**

[![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=Streamlit&logoColor=white)](https://streamlit.io)
[![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)](https://python.org)

Dashboard interactivo para análisis de datos de ventas B2B, combinando habilidades técnicas con inteligencia comercial.

## 🎯 Características

- 📊 **Análisis visual interactivo** de ventas
- 🤖 **Procesamiento automático** de datos CSV
- 📈 **Métricas clave** y tendencias
- 🎯 **Segmentación** por producto, región, vendedor
- 🔗 **Integración** Google Sheets (próximamente)

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python, Pandas, Plotly
- **Data:** CSV, Google Sheets API
- **Deployment:** Streamlit Cloud, Heroku
